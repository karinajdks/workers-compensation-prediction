# Проект: Прогнозирование стоимости страховых выплат

## Описание проекта

Цель проекта – разработать модель машинного обучения для предсказания итоговой стоимости страхового возмещения (UltimateIncurredClaimCost) на основе характеристик работника и параметров случая.

Используется датасет **Workers Compensation** (ID: 42876) из библиотеки OpenML, содержащий 100,000 реальных записей о страховых случаях.

## Бизнес-ценность

- Точное прогнозирование страховых выплат
- Оптимизация формирования резервов
- Улучшение тарифной политики
- Выявление факторов риска

## Технологии

- Python 3.9+
- Streamlit (интерфейс)
- Scikit-learn (ML модели: Linear Regression, Ridge, Random Forest, XGBoost)
- Pandas/NumPy (обработка данных)
- Matplotlib (визуализация)

## Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/karinajdks/workers-compensation-prediction.git
cd workers-compensation-prediction
```

### 2. Создайте виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустите приложение

```bash
streamlit run app.py
```

## Результаты

| Модель | R² | MAE | RMSE |
|--------|-----|-----|------|
| Linear Regression | 0.0925 | $15,966 | $46,984 |
| Ridge Regression | 0.0925 | $15,966 | $46,984 |
| Random Forest | 0.0199 | $17,027 | $48,827 |
| XGBoost | 0.0367 | $15,339 | $48,405 |

**Лучшая модель:** Linear Regression (R² = 0.0925)

**Вывод:** Низкий R² объясняется отсутствием в датасете ключевых факторов (тяжесть травмы, качество лечения, судебные издержки). Проект демонстрирует полный цикл разработки ML-решения.
